কী কী কাজ করা যায়

West Bengal ration card services-এর মধ্যে নতুন আবেদন, family member update, category change, application status, ration card download, Aadhaar seeding এবং dealer/FPS সম্পর্কিত তথ্য থাকে। কিছু service login/OTP ভিত্তিক, তাই পরিবারের যে সদস্যের Aadhaar ব্যবহার করছেন তার mobile link থাকা জরুরি হতে পারে।

প্রয়োজনীয় ডকুমেন্ট

  • Aadhaar / PAN / Voter ID প্রয়োজন অনুযায়ী
  • Mobile number এবং email ID
  • Address proof বা identity proof
  • Bank passbook / photo / certificate প্রয়োজন অনুযায়ী
  • Application acknowledgement number থাকলে status check সহজ হয়

সতর্কতা

Ration status check করার সময় Aadhaar OTP লাগতে পারে। private বা unknown website-এ Aadhaar number দেওয়া উচিত নয়। Official WBPDS/Food & Supplies portal ব্যবহার করুন।

FAQs

Ration status কী দিয়ে check করা যায়?

Official portal-এ application number, ration card number বা Aadhaar based login option থাকতে পারে।

e-Ration card download করা যায়?

হ্যাঁ, WBPDS official portal-এর E-Card Download page (উপরের button-এ দেওয়া আছে) থেকে সরাসরি e-Ration Card download করা যায়।

Aadhaar seeding দরকার কেন?

DBT/food benefits এবং family verification-এর জন্য Aadhaar seeding ব্যবহার করা হতে পারে।
